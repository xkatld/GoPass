# GoPass - 密码管理器

简单的个人密码管理工具，使用 Go + TailwindCSS 开发。

## 功能
- 密码存储和管理
- 密码生成器
- 数据导入/导出
- 分类管理

## 快速开始

需要 Go 1.21+

```bash
chmod +x start.sh
./start.sh
```

然后访问: http://localhost:8080

## 使用
1. 注册账户
2. 登录系统
3. 添加和管理密码
